# MyFinance – Personal Expense Tracker App

## Overview
MyFinance is a modern Android application for tracking personal income and expenses.  
It allows users to manage daily finances, categorize transactions, and view monthly or daily reports.  
The app is built using Kotlin and Jetpack Compose with MVVM architecture and Room Database.

---

## Features
- Add, edit, and delete income & expenses
- Categorize transactions (Food, Transport, etc.)
- View real-time balance updates
- Monthly and daily summary charts
- Modern UI with Material Design 3
- Offline data storage using Room Database
- Clean MVVM architecture for easy scalability
- Demo APK included

---

## Requirements
- Android Studio (latest version)
- Kotlin 1.9+
- Gradle 8+
- Minimum SDK: 24+
- Internet connection (optional, for demo video)

---

## Installation Instructions
1. Download and unzip the file.
2. Open the project in Android Studio.
3. Let Gradle sync automatically.
4. Run the app on emulator or physical device.
5. (Optional) Change the package name and app icon before publishing.
